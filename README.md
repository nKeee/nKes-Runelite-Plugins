## NPCHighLight
- Repeatedly draws a dot inside NPCs hitbox
- Antiban to randomize the dot positon almost every frame
- Customizable dot color

## ScreenMarkers
- Draws a random dot within the chosen area
- Antiban to randomize the dot positon almost every frame
- Customizable dot color
